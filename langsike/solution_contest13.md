 ## div2 A
 ### Problem description
 > 以字符串形式给出一个杠杆的形态(支点位置，重物位置及重物质量)，求杠杆是平衡，左倾还是右倾。(字符串长度3≤|s|≤1000000)。
 ### Solution
 由杠杆平衡条件可以得到,设sum=∑(xi-k)wi,xi为第i个重物的位置，wi为第i个重物的质量，若sum<0杠杆左倾，sum>0杠杆右倾，sum=0杠杆平衡。
 ```cpp
 #include <bits/stdc++.h>
 using namespace std;
 int main () {
     printf("I want AC\n");
 }
 ```
